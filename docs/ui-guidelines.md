# UI 规范

## 页面结构

- 顶部固定为学院导航。
- 总览页按学院分组展示科目。
- 科目页固定包含：统计面板、模式切换、筛选栏、题目卡。
- 学习、错题、收藏、模拟考试模式显示右侧答题卡。
- 题库浏览不显示右侧答题卡。

## 视觉规则

- 背景使用浅灰。
- 内容卡片使用白底、1px 边框、8px 到 16px 圆角。
- 主操作使用克制蓝色。
- 不使用大面积渐变和装饰性图形。
- 大题库场景下，右侧答题卡默认只显示当前题附近题号，不能把全部题号铺在主界面。
- 完整答题卡放入抽屉或弹窗，允许滚动、筛选和搜索。

## 交互规则

- 所有科目都必须显示准确率。
- 选项提交后显示正确/错误状态和解析。
- 主观题提交后只记录已练，不自动判错。
- 收藏和错题必须刷新后保留。
- 学习模式提交前不暴露正确答案。
- 错题复习和待复习进入题目时不直接显示上次答案。
- 提交后可以保存 10 个字左右的主动回忆提示，下次复习先看自己的提示。
- 正确、错误、收藏、待复习状态不能只依赖颜色，需要同时使用文字、角标或边框。
